Smart Crop Recommendation System 🌱

The Smart Crop Recommendation System is a web-based platform that helps farmers and agricultural experts identify the most suitable crop to grow based on key soil parameters like Nitrogen, Phosphorus, Potassium, pH, temperature, humidity, and rainfall.

Built using Django and integrated with machine learning algorithms like Random Forest, Decision Tree, and KNN, this system improves yield and promotes sustainable farming.

The system encourages smart agriculture by providing accurate crop recommendations, which helps maximize output, reduce crop failure, and protect environmental resources.

🔧 Technologies Used

  • 🐍 Python & Django Framework
  • 🧠 Machine Learning (Random Forest, KNN, Decision Tree)
  • 💻 HTML, CSS
  • 📊 Scikit-learn, Pandas, NumPy
  • 📝 Django Forms & Model Integration

📈 Features

  • 🌾 Crop Prediction based on Soil Parameters
  • 📊 Data Visualization of Soil Conditions
  • 🔍 User-friendly Interface for Easy Navigation
  • 📚 Educational Resources on Smart Agriculture
  • 📱 Responsive Design for Mobile and Desktop
